# Calculator Project

This project is a simple calculator built using HTML, CSS, and JavaScript. It was created to practice front-end web development concepts such as user interface design, event handling, and DOM manipulation. The calculator provides a clean and responsive interface for performing basic arithmetic calculations.

## Features

- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
  - Percentage (%)
- Clear the entire display
- Delete the last entered character
- Keyboard input support for faster calculations
- Instant calculation results
- Responsive and user-friendly design
- Smooth button animations and modern styling

## Technologies Used

- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for calculator functionality and user interactions

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Run the `index.html` file in any modern web browser.
4. Use the calculator to perform calculations.
